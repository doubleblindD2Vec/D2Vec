In this experiment - API example code search (text-to-code retrieval), our goal is to retrieve the API code examples given an English query. For D2VEC, we treated the query and a code example as two paragraph descriptions with different topic IDs and used the trained model to derive their vectors.

Data Collection

- Training Data: We only included the documentation of JDK APIs Download
- Retrieval Data: To evaluate the performance of our retrieval techniques, we collected a parallel dataset of queries and API code examples from a Java tutorial website, named KodeJava Download

Result

The retrieval accuracy is 19.9-41.2% with one to five suggestions.
In comparison to Ye et al.[1], the improvement of 8.2% (70% relatively) and 11.7% (39.6% relatively) in top-k accuracies are achieved by D2VEC in the code search application:
