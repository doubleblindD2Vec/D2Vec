In this experiment - API mapping between libraries (code-to-code retrieval), our goal is to derive single mappings for equivalent APIs between two libraries, JDK and Apache.

Data Collection

To train D2VEC, we used the training data with both API documentations of Java JDK Core and Apache Commons libraries.
